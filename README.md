# NYC Rats in Restaurants
A dashboard that lets users know what restaurants in New York City failed rodent inspections updated at the beginning of the week. The map shows each the names and locations of each restaurant that has failed inspection.

The dashboard is currently hosted using Streamlit [here](https://nyc-rats-in-restaurants.streamlit.app/).

Source data is from New York City's Open Data platform [here](https://data.cityofnewyork.us/Health/Rodent-Inspection/p937-wjvj/about_data).
